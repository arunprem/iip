# IIP Mobile (`iip_app`)

Flutter client for IIP (iOS and Android).

## App icon

Launcher icons use the Kerala Police emblem (`assets/images/kerala_police_logo_opaque.png`).

Regenerate after updating the logo:

```bash
python3 tool/generate_launcher_icons.py
python3 tool/generate_splash_assets.py
# or: dart run flutter_launcher_icons
```
